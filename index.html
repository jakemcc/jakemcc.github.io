Projects by Jake McCrary

My technical blog.

Default Equipment

Tiny little web service that consumes Strava webhooks and changes my rides categorized as electric bicycle rides to using my one wheel as the equipment.

Photo Fit

Tiny web tool that intended to be used on your phone. You select a photo from your phone and it resizes it and adds top and bottom black bars to it to make it fit your phone better. Intended to be used to make properly sized background images.

Book Robot

Project that takes your goodreads.com reading list and searches libraries and Amazon for your to-read books.

Dune Dictionary

A custom Kindle dictionary for the Dune series.

Functional JavaScript Companion

Instructions on how to download the Kindle dictionary I made for the excellent book Functional JavaScript. It maps the names of functions defined in the book to their implementations.

lein-autoexpect

Leiningen plugin to automatically run your expectations when files in your project change.

lein-test-refresh

Leiningen plugin to automatically run your clojure.test tests when files in your project change.
